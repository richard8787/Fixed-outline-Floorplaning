# Fixed-outline-Floorplanning
Implement fast simulated annealing to do fixed-outline floorplanning

## Compile and Execute

- Compile
```
make clean
make
```

- Run
```
./fp <alpha value> <input_block file> <input_net file> <output_file>
```
